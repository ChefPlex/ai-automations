# Director Review Prompts

These prompts are written for director level TPMs who need to review programs, coach TPMs, challenge assumptions, and improve executive artifacts.

## Recommended usage

Use these prompts to:

- Review a program plan before leadership review
- Improve an executive update
- Challenge a senior TPM strategy
- Identify hidden dependencies
- Build an operating model
- Improve metrics
- Prepare a retrospective

## Director principle

The director level TPM should use AI to improve leverage: better questions, better operating models, better risk visibility, and stronger communication across teams.
