# Director Review Prompts

These prompts are written for director-level TPMs who need to review programs, coach TPMs, challenge assumptions, and improve executive artifacts.

This is where AI is most useful as a second set of eyes. It can help spot missing owners, weak metrics, soft risk language, unclear decisions, and plans that look fine until the first dependency slips.

## Good use cases

- Reviewing a program plan before leadership review
- Improving an executive update
- Challenging a senior TPM strategy
- Finding hidden dependencies
- Building an operating model
- Improving metrics
- Preparing a retrospective

## Director principle

The director-level TPM should use AI to ask better questions, create better operating models, improve risk visibility, and help teams communicate more clearly across functions.

The model is not the leader. It is the reviewer who never gets tired of asking whether the owner, date, metric, and decision are actually clear.
