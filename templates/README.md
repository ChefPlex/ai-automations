# Templates

These templates help TPMs collect structured inputs before running prompts.

They are intentionally plain. The goal is not beautiful formatting. The goal is to gather the minimum useful context so the prompt output is accurate, safe, and easy to validate.

## Included templates

- Weekly status input template
- Salesforce signal intake template
- Jira export field list

## Usage

Copy the relevant template into a Google Doc, Google Sheet, or note. Fill in only approved and sanitized information. Then use the completed template as input for the matching prompt or workflow.

PS: A template is doing its job when it makes missing information obvious before the meeting, not during it.
