# Examples

This directory contains sanitized fictional examples showing how outputs could look.

The examples are intentionally generic. They should not be treated as real company data, customer data, legal advice, security advice, or compliance guidance.

## Examples included

- Weekly status example
- Escalation example
- Decision memo example
- Launch readiness example

## How to read them

Look at the shape, not the invented facts. A good example should show clear context, plain risk language, named owners, and a next step. It should not sound like the program is healthier than it is.
