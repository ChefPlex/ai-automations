# Examples

This directory contains sanitized fictional examples showing how AI-assisted TPM workflow outputs could look.

The examples are intentionally generic. They should not be treated as real company data, customer data, legal advice, security advice, compliance guidance, or proof that a generated output is correct without human review.

## Review note

A good example should show uncertainty, not hide it. If the input is missing owners, dates, evidence, or decision context, the output should say that plainly.
