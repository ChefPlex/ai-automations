# Workflows

This directory contains end to end TPM workflows that combine prompts, source systems, review steps, and publishing guidance.

## Workflows included

- Weekly status workflow
- Customer escalation workflow
- Launch readiness workflow
- Compliance intake workflow
- Executive review workflow

## Workflow pattern

Each workflow follows this structure:

1. Goal
2. Source systems
3. Inputs to gather
4. Safety checks
5. Prompt sequence
6. Human review checklist
7. Output artifacts
8. Systems to update

## Reminder

A workflow is not complete until the right system of record is updated.
